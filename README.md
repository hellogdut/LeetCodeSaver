# LeetCodeSaver 

Download your AC codes from Leetcode

## Install Dependences

pip3 install lxml

pip3 install requests

## Usage
fill in your account of Leetcode
```
username = 'xxx'
password = 'xxx'
```
and then start
```
downloader = LeetcodeSaver(username,password)
downloader.start()
```
the codes will be download at "/Solutions" under the excute path
